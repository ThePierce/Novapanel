# Nova Panel documentatie

Nova Panel start leeg zodat iedere installatie een eigen dashboard kan opbouwen.

1. Start de add-on.
2. Open Nova Panel via de Home Assistant zijbalk.
3. Zet edit mode aan.
4. Voeg kaarten toe aan de sidebar of aan hoofdsecties.

De dashboardinrichting wordt buiten de repository opgeslagen. Commit daarom geen `.env`, `.data`, exports of persoonlijke Home Assistant-entiteitsnamen.
